# Custom Game Engine

This is a custom game engine project I started working on January 2025. I don't have a functional game working yet, but I currently have plans to finish it in graduate school if I don't get a job this next spring.

I built this engine in C++11 utilizing OpenGL as the main rendering API. I am currently working on reworking the object system to work with a custom ECS data-oriented design pattern. After I finish that, I plan to make a simple game and then implement Vulkan as an alternate rendering API.
