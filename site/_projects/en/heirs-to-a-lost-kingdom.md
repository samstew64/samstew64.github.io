# Heirs to a Lost Kingdom

This was a game proposal for my university's year-long game project. I made a 9-page design document, created a 5-minute pitch presentation, and wrote a dynamic split screen algorithm with an example in Unreal Engine for a proof of concept. I have both the design document and a PDF-export of the presentation slides in this folder.

This project proposal is finished.
