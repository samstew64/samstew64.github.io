# Dragon Kisser

⼤学のプログラムの⼀年間のビデオゲームプロジェクトで２０２５年４⽉に始まり、２０２６年４⽉に終わる予定です。Unreal Engine 5を使⽤し、デザイナーとプログラマも含まれるチームは約３０⼈の⼤学⽣がいます。

プロジェクトは開発中で、２０２６年４月に終える予定です。

<a href="https://www.youtube.com/watch?v=vEWujqb0gKo" class="youtube-link-button">▶「BYU Universe」に行われた面接をご覧</a>
