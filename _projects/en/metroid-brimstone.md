# Metroid Brimstone (school project & fan game)

I led a group of 4 students with a goal to make a video game in Unreal Engine 5 in less than 3 months. The other students had very limited experience in Unreal, and so it was my job to figure out a game idea, make a plan to fulfill each individual students' goals, get everybody up to speed on how to use Unreal, and lead the team to make a fully finished product.

This project is finished.

## Responsibilities

#### My role: Director/Producer

### Things I did:

- Gameplay programming for player character.
- AI Systems
    - Built one of the enemy character's AI and the base enemy AI system.
    - Taught the other students to make the rest of the AI.
- Version Control
    - Learned how to setup and manage a Perforce depot.
- Rigging
    - 4x custom character rigs
    - Modified main character rig to work in a game engine.
- Animation
    - Animated every character (~50 animations)
    - Created the animation system for the main character and boss.
- Modeling and Texturing
    - Created a procedural modeling pipeline to make modeling quicker and easier for the whole team.
    - I created the base sphere model of the boss and crawling enemy.
- Developed a Blender to Unreal asset pipeline.

### Things I did not do:

- Game mode/instance logic (game progression)
- Half the enemy AI (including the boss)
- Player character model and base rig (we got it online).
- Base sphere models for the flying and lava enemies.
- Level layout
- Environment art
- VFX
- Music
- SFX
